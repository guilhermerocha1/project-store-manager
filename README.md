# Project Store Manager

# Contexto
Este projeto faz um CRUD em API ou seja, cria novos dados, ler dados, atualiza e deleta dados, a partir de um banco MySQL.

## Técnologias usadas

Back-end:
> Desenvolvido usando: NodeJS, ExpressJS, MYSQL, ES6


## Instalando Dependências

> Backend
```bash
npm install
``` 

## Executando aplicação

* Para rodar o back-end:

  ```
  cd src/ && npm start
  ```

## Executando Testes

* Para rodar todos os testes:

  ```
    npm test
  ```
